# CSCE321Project-FileOrganizer
Simple file organizer.

I set it up in a way that has the main, config, logger, and organizer scripts seperate from one another. This is just for easier coding so we don't have to dig through a single giant script in order to make changes.

Everything is set up, all you need to do is input a folder path in the config script for the source and destination. Remember to use forward slashes (/) instead of back slashes (\). Some things I want to work on is making an in menu prompt to change the source and destination directory within the program, I would also like to create a gui interface if possible so it looks cleaner overall.
